# V21 changelog

Released: 15 July 2026

- Added automatic Unpaywall lookup for all publication DOIs.
- Added `Open Access PDF` only when Unpaywall provides a direct legal PDF URL.
- Added `Open Access Version` when a legal OA landing page exists without a direct PDF.
- Non-OA publications do not display an OA button.
- Added a compact Share action to each publication.
- Share uses the device's native share interface when supported.
- Desktop fallback includes Copy link, Email, LinkedIn, X (Twitter), and Facebook.
- Shared URLs point to a stable DOI-derived anchor and highlight the selected publication.
- Added weekly and manual GitHub Actions updates for `data/unpaywall.json`.
- Updated website version to V21.
